    2. Python#

    Python é uma linguagem de programação de alto nível, interpretada, de script, imperativa, orientada a objetos, funcional, de tipagem dinâmica e forte. Foi criada por Guido van Rossum e sua primeira publicação foi em 1991. Python é uma linguagem que enfatiza a legibilidade do código, permitindo que programadores expressem conceitos em menos linhas de código do que seria possível em linguagens como C++ ou Java.

    1. Fundamentos e Funcionamento#

    • Definição técnica: Python é uma linguagem de programação que suporta múltiplos paradigmas de programação, incluindo procedural, orientado a objetos e funcional.
    • Como funciona internamente: Python é interpretado, o que significa que o código é executado linha por linha, facilitando a depuração e tornando-o ideal para prototipagem e outras tarefas de programação rápida. O Python usa um coletor de lixo para gerenciamento de memória automático.
    • Componentes principais: Incluem o interpretador Python, a biblioteca padrão que acompanha o interpretador, e muitos módulos e frameworks disponíveis para tarefas específicas, como Django para desenvolvimento web e Pandas para análise de dados.

    2. Estrutura Completa do Tema (Nível Prova)#

    • Regras:
      • Identação é obrigatória para blocos de código.
      • Nomes de variáveis são case-sensitive.
      • Comentários iniciam com #.
    • Classificações:
      • Tipos de dados: Numéricos (int, float), Sequências (string, list, tuple), Mapeamentos (dict), Conjuntos (set).
      • Estruturas de controle: if, for, while, try-except.
    • Categorias:
      • Tipos de funções: Definidas pelo usuário, Anônimas (lambda), Incorporadas (built-in).
      • Módulos e Pacotes: Organização de código em módulos e pacotes para reutilização.
    • Exceções:
      • Python possui um conjunto de exceções integradas, como ValueError, TypeError, que podem ser personalizadas.

    3. Regras Técnicas e Comportamento#

    • Funcionamento detalhado: O Python utiliza o PEP 8 como guia de estilo para a escrita do código, promovendo uma sintaxe limpa e legível.
    • Limitações:
      • Devido à sua natureza interpretada, pode ser mais lento que linguagens compiladas como C.
      • Gerenciamento de memória automático pode levar a um consumo maior de recursos.
    • Condições de uso: Ideal para desenvolvimento rápido, prototipagem, scripts, análise de dados, aprendizado de máquina, desenvolvimento web, entre outros.

    4. Diferenças Críticas#

    • Comparação com outras linguagens:
      • Python vs. Java: Python é interpretado, enquanto Java é compilado para bytecode e executado na máquina virtual Java (JVM). Python geralmente permite um desenvolvimento mais rápido devido à sua sintaxe concisa.
      • Python vs. C: Python é de alto nível e gerencia a memória automaticamente, enquanto C é de baixo nível e requer gerenciamento manual de memória.

    5. Exemplos e Aplicações#

    # Exemplo de função em Python

    return a + b

print(soma(5, 3))
    • Cenários reais: Automação de tarefas do dia a dia, desenvolvimento de aplicações web com Django ou Flask, análise de dados com Pandas e NumPy, criação de modelos de aprendizado de máquina com scikit-learn ou TensorFlow.

    6. Pegadinhas de Prova#

    • Erros comuns: Confundir a necessidade de indentação com outros estilos de blocos de código.
    • Afirmações que parecem corretas: “Python é uma linguagem compilada” - Errado, Python é interpretado.
    • Termos que invalidam questões: Afirmar que Python não suporta programação orientada a objetos.

    8. Simulado (Certo ou Errado)#

    1. Python é uma linguagem de programação compilada. (Errado)
    2. A indentação em Python é utilizada para definir blocos de código. (Certo)
    3. Python não suporta programação orientada a objetos. (Errado)
    4. Funções lambda em Python são usadas para criar funções anônimas. (Certo)
    5. Python realiza gerenciamento de memória manualmente. (Errado)
    6. Python permite a sobrecarga de operadores. (Certo)
    7. Módulos em Python não podem ser reutilizados em outros programas Python. (Errado)
    8. Python é menos adequado para desenvolvimento web do que PHP. (Errado)

    Gabarito comentado:#

    1. Python é interpretado, não compilado. O código é executado diretamente pelo interpretador.
    2. Certo. A indentação é fundamental em Python e é usada para delimitar blocos de código.
    3. Errado. Python suporta plenamente a programação orientada a objetos, permitindo a definição de classes, herança, e polimorfismo.
    4. Certo. Funções lambda são uma forma concisa de declarar funções anônimas para uso em expressões.
    5. Errado. Python utiliza um coletor de lixo para gerenciar a memória automaticamente.
    6. Certo. Python permite a sobrecarga de operadores, permitindo que objetos de classes definidas pelo usuário suportem operadores como +, -, *, etc.
    7. Errado. Módulos e pacotes em Python são projetados para serem reutilizáveis, facilitando a organização e compartilhamento de código.
    8. Errado. Python é amplamente utilizado para desenvolvimento web, com frameworks robustos como Django e Flask, e é tão capaz quanto PHP nesta área.
